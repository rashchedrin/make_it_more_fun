# Cosmic Dodge

A single-file HTML5 game. Dodge falling asteroids as long as you can.

## How to play

- **Move:** Arrow Left/Right or A/D
- **Start:** Click "Start Game" or press Space/Enter
- **Goal:** Survive. Score increases over time and when asteroids leave the screen. You have 3 lives; each hit costs one. Game gets harder as spawn rate increases.

## Run locally

1. Open `index.html` in a browser (double-click or File → Open), or
2. Serve the folder and open in browser, e.g.:
   ```bash
   python3 -m http.server 8765
   ```
   Then open http://localhost:8765

## Upload as HTML page

Upload `index.html` to any static host (GitHub Pages, Netlify, etc.). No other files or server logic required.
